# cli-toolbelt

A CLI toolbelt comprised of multiple subcommands

## Help

```text
A CLI tool for IT related tasks

Usage: cli-toolbelt <COMMAND>

Commands:
  cidr    Calculate and display info on CIDR
  dns     Query DNS records
  ssl     Establish a SSL TCP connection and display SSL certificate
  rng     Generate a random number from a range
  tcp     Create a TCP connection (like telnet)
  ports   Scan host(s) TCP port(s)
  domain  Generate an overview of a domain
  diff    Compare the lines of two files
  help    Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
